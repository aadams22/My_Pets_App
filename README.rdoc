== Oli & Olive


=== Description:
Oli and Olive is a landing page where users can choose either a cat or a dog to sit as their landing page pet and tell them the current weather.

##Tech Used:
* Ruby
* Rails
* Devise
* ForecastIO

=== Future Implementations:
* Allow for pets to have a hunger scale and allow users to feed their pet
* Allow for pets to have a love scale and allow users to pet their pet
* Create chart to show love or hunger need
